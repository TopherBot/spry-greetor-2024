# Spry Greetor 2024

A **tiny** command‑line utility written in Python that says hello.

## Features
- No external dependencies (just the Python standard library).
- Works on any platform with Python 3.7+.
- Accepts an optional name to personalize the greeting.

## Installation
```bash
# Clone the repository (or copy the single file)
git clone https://github.com/your‑username/spry-greetor-2024.git
cd spry-greetor-2024

# (Optional) make the script executable and add to PATH
chmod +x greet.py
sudo mv greet.py /usr/local/bin/greet
```

## Usage
```bash
# Default greeting
python greet.py
# or, if installed as "greet"
greet

# Personalized greeting
python greet.py Alice
# or
greet Bob
```
